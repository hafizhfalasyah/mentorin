<h1 align="center"><b>🚀 Mentorin Education — CBT Simulation Platform</b></h1>

---

## 📌 Overview

**Mentorin Education CBT Simulator** is a comprehensive web-based platform designed to facilitate computer-based test (CBT) simulations, tryouts, and selection exams for candidates preparing for law enforcement, military, and civil service recruitment.

The platform provides a realistic exam environment that closely replicates actual test conditions, including time constraints and interface design. It also integrates a full-featured learning system, allowing users to track progress, review performance, and access structured study materials.

Developed for **mentorineducation.id**, this system demonstrates the implementation of scalable web architecture, secure authentication, and seamless online payment integration.

---

## 🎯 Key Highlights

- Realistic CBT simulation environment (exam-like interface & timing)  
- Integrated learning ecosystem (materials, practice, and evaluation)  
- Automated payment system using Midtrans  
- User performance tracking and analytics  
- Built with modern Laravel ecosystem (Laravel 11 + Breeze)  

---

## ✨ Features

- 🔐 Authentication System (Laravel Breeze)  
- 👥 Multi-user access system  
- 📝 CBT Simulation with real-time timer  
- 📊 Progress tracking and performance analytics  
- 📚 Practice questions with explanations  
- 🗂️ Test history and score recap  
- 📖 Structured learning materials  
- 💳 Secure Online Payment (Midtrans Integration)
